Driving Vehicle License Mangment System (DVLD)

Driving Vehicle License (DVLD) is an educational desktop-baed managment system designed to automate and manage the complete lifecycle of driving license operations.
The system is inspired by real-world traffic and licensing adminstrations and accuracy validation, role-based control, and scalability.

This project is built as a learnning-by-doing system to strengthen skills in:
C# (.NET)
SQL Server
Database design
Desktop application architecture
Real-worl business logic

Features
Person Management
Add, update, delete, and serch people
National ID (CIN) uniqueness validation
Gender, date of birth, and contact information handling

License Managment
Issue new driving licenses
Renew licenses
Replace lost or damaged licenses
License status tracking (Active/Expired/Suspended)

Application System
License application creation
Application status tracking
Applicationfees management

Test Management
Vision test
Written test
Practical driving test
Test scheduling and results recording

System Architecture
Presentation Layer:
Windows Forms (WinForms)

Business Logic Layer (BLL):
Validation rules, workflow logic, and domain services

Data Access Layer (DAL):
SQL Server with stored procedures and parameterized queries

Database:
Normalized relational schema
Foreign keys and constraints  
Views for reporting and filtering
